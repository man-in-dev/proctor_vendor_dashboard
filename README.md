# Vendor Dashboard

A Next.js vendor portal dashboard application.

## Features

- Vendor Portal layout with sidebar navigation
- Profile page for managing vendor information
- Modern UI with Tailwind CSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with pages and layouts
- `/components` - React components (Sidebar, TopBar)
- `/public` - Static assets

## Pages

- `/` - Redirects to profile page
- `/profile` - Vendor profile management page
